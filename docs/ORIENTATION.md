# Ориентация по проекту (ЕГЭ Roadmaps)

Этот документ — “карта местности”, чтобы быстро понимать, где что лежит и как проект работает.

## Что это за сайт

Astro-сайт, который показывает дорожные карты подготовки к ЕГЭ:
- по предметам (URL `/<subjectId>`, например `/math`)
- по уровню (query `?tier=base|standard|hard`)

Данные роадмапа генерируются на сервере, а прогресс “Изучено” хранится локально в браузере.

## Главный поток данных (как всё собирается)

1) Страница `src/pages/[roadmapId]/index.astro`:
   - берёт `roadmapId` из URL
   - берёт `tier` из query
   - вызывает `generateRoadmap({ subjectId: roadmapId, tier })`
   - рендерит фазы и темы

2) Генератор `src/lib/roadmaps/generator.ts`:
   - знает уровни `base/standard/hard` и их метаданные
   - выбирает контент предмета из `src/lib/roadmaps/subjects/*`
   - если предмет не подключен — строит “заглушки” (чтобы UI не ломался)

3) Контент конкретного предмета:
   - каждый предмет — отдельный файл в `src/lib/roadmaps/subjects/`
   - в файле задаются фазы (разделы) и темы + ссылки по теме

4) Прогресс:
   - хранится в `localStorage`
   - ключ: `ege-roadmap:${roadmapId}:${tier}`
   - значения: массив `topic.id`, отмеченных как “Изучено”

## Где править что (быстрые ответы)

- Добавить предмет в меню:
  - `src/lib/subjects.ts` (добавить `{ id, title }`)

- Добавить роадмап для предмета:
  1) Создать файл `src/lib/roadmaps/subjects/<subjectId>.ts`
  2) Экспортировать `export const <name>Roadmap: Record<RoadmapTier, RoadmapPhase[]> = ...`
  3) Подключить в `src/lib/roadmaps/generator.ts` (import + добавить в `subjectRoadmaps`)

- Поменять UI “карты” и боковой панели:
  - `src/pages/[roadmapId]/index.astro`

- Поменять стили таймлайна/панели:
  - `src/styles/global.css`

## Соглашения по данным роадмапа

- `phase.id`: строковый идентификатор фазы (может быть любым, но лучше стабильным).
- `topic.id`: идентификатор темы; используется для клика в UI и для сохранения прогресса.
  - Если изменить `topic.id`, пользовательский прогресс для этой темы “сбросится”.

## API для отладки/интеграций

Есть API-роут:
- `/<roadmapId>.json?tier=...` → JSON текущего роадмапа

Файл: `src/pages/[roadmapId].json.ts`.
